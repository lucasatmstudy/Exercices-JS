Setup d’un projet JavaScript avec Vite

Découvrez comment configurer rapidement un projet JavaScript moderne en utilisant Vite, un outil de build ultra-rapide et simple à utiliser.

⚡ Qu’est-ce que Vite ?

Vite est un bundler moderne qui permet de développer des applications JavaScript ou TypeScript avec une rapidité exceptionnelle grâce à son serveur de développement basé sur ESM (ECMAScript Modules).

  • Démarrage instantané du serveur de dev.
  • Hot Module Replacement (HMR) ultra-rapide.
  • Compatible avec Vue, React, Svelte, etc.
  • Support natif de TypeScript et PostCSS.

🛠️ Installation de Vite

Assurez-vous d’avoir installé Node.js (version 18 minimum recommandée). 

                # Créer un projet avec npm
npm create vite@latest

# ou avec pnpm
pnpm create vite

# ou avec yarn
yarn create vite

Choisissez ensuite :

  • Nom du projet (ex: mon-projet)
  • Framework (ex: Vanilla, Vue, React...)
  • Langage (JavaScript ou TypeScript)

📂 Structure d’un projet Vite

Voici la structure de base générée par Vite : 

                mon-projet/
├── index.html
├── package.json
├── vite.config.js
├── /node_modules
└── /src
    ├── main.js
    └── style.css

Le fichier main.js est le point d’entrée de l’application. Vous pouvez y importer votre CSS et vos modules JS.

🚀 Lancer le projet

Utilisez les commandes suivantes :

                # Installer les dépendances
npm install

# Lancer le serveur de dev
npm run dev

# Build du projet
npm run build

Par défaut, le serveur de développement est accessible à http://localhost:5173

✅ Bonnes pratiques

  • Versionner package.json et vite.config.js.
  • Ne pas versionner node_modules/ (utiliser un .gitignore).
  • Utiliser .env pour stocker les configurations sensibles.
  • Organiser son code dans des sous-dossiers /components, /utils, etc.
